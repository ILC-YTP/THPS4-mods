This is less of a "mod" and more of a just replacing a flawed DLL with a better new one

The Stutter that happens mainly when music or audio streams are paused, resumed, or sometimes called is due to the game waiting on it's BINK video dll to finish up.

Replacing it with a newer version seems to help a lot

The 7z file here contains the DLL I replaced it with

The 7z file is also password protected, but that's mainly to avoid browsers and AV's from false flagging it

If you would rather get it from the exact source, go here: https://github.com/vogonsorg/radgametools/blob/main/binkw32/1.9z-binkw32.dll

## Installation
Backup `binkw32.dll`, then place the new `binkw32.dll` into THPS4's `Game` folder (NOT the `data` folder)
If needed, rename `1.9z-binkw32.dll` to `binkw32.dll`
