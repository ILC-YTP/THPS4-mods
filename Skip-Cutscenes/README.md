The mod I'm sure the vast majority of you want

Here is a list of changes this mod makes:

- All goal ending cutscenes can now be skipped. You no longer have to wait for the text and UI pop-ups to disappear.
- Gaining Stat points no longer opens the stats menu automatically, requiring you to use statpoints in the pause or character menu.
- Save game prompt after every goal removed.
- Ability to skip the intro cutscene once ped starts talking
- Goals SF:Feed the 4 Angry Sea Lions, SF:Shark Cutscene, SH:Gap Across the Suspended Containers, UK: Free Stompy and ZOO:Elephant Hop have additional intro/mid cutscenes, all of which can now be skipped
- Goals ALC: King of Kona and ALC: Bullseye Launch have a new option to trigger the goal to win/lose depending on if you have the required points
- Shortened the amount of time waiting until Half Pipe ramp starts moving in Tony Hawk's Pro Goal
- Game no longer pauses when triggering camera scripts for Put Kong Back in His Cage goal

Misc Changes:

- Removed some un-needed functions where some goals would stop playing music, causing a stutter (with default binkvideo dll)
- Lowered the volume of penguins screeching when falling down

What wasn't changed:

- The 1.5 Second delay before a goal starts (this can easilly be changed, but I kept it for personal preferences and to prevent confusion on losing context of a goal if started too early. All Gen 6 games afterwards have this delay as well)
- Did not remove the cam animation for Tony Hawk's Pro Challenge when the ramp moves
- Did not remove the cam animation for Bob's Pro Challenge when the loop moves
- Did not make the cam animations skippible for Collect 6 souvenir photos (they're short enough as is really)
- Messages about unlocking new levels, specific pro challenge unlocked, unlocking all pro/harder challenges, and beating all goals and gaps remain and don't get skipped.

### Installation: 
Drag and drop the data folder into THPS4's Game folder, then replace the files when asked. Feel free to back up your older install/data folder
