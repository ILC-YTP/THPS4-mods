# THPS4-mods
A list of mods I made for Tony Hawk's Pro Skater 4, including one to skip all cutscenes


### Huge thanks to [DCxDemo](https://github.com/DCxDemo) and his tool [ThpsQScriptEd](https://github.com/DCxDemo/LegacyThps). Without it, most of these mods wouldn't have been made.




# Mod list

## [Skip Cutscenes](https://github.com/ILC-YTP/THPS4-mods/tree/main/Skip-Cutscenes)
Finally, no more waiting like 8 to 16 seconds at the end of every goal and being asked to save the game every time, or having to wait during goals like feeding the seals.

## [Faster Save/Load](https://github.com/ILC-YTP/THPS4-mods/tree/main/Faster-SaveLoad)
Remove the forced 2 seconds of wait time when loading or saving to THPS4 save files.

## [Fix Game Stutters When Pausing Game](https://github.com/ILC-YTP/THPS4-mods/tree/main/Fix-Stutter)
Fix an issue where pausing and resuming stream and music files would cause the game to stutter/freeze for a few frames.
